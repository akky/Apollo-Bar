A simple announcements plugin.

== Description ==

The Apollo Bar is a really simple announcements and notification plugin. Used with Custom Post Type to publish announcements and cycle plugin to rotate messages with fade animation. There is a very simple administrative interface. Can be switched off and on the Apollo Bar and author logo, or you can choose the 3 color scheme. Available in 2 languages, English and Hungarian. More translations are welcome.

== Installation ==

1. Upload `apollo-bar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add announcements to the Announcements Custom Post Type menu.
4. Enable the plugin in the 'Settings' => 'Apollo Bar'. Well done!
