A simple announcements plugin.

== Description ==

The Apollo Bar is a really simple announcements plugin. Used with Custom Post Type to publish announcements and cycle plugin to rotate messages with fade animation. There is a very simple administrative interface. Can be switched off and on the Apollo Bar and logo, or you can choose the color scheme. Available in 2 languages, English and Hungarian. More translations are welcome.

== Installation ==

1. Upload `apollo-bar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add announcements to the Announcements Custom Post Type menu. Well done!

== Frequently Asked Questions ==

= How to enable Apollo Bar =

Go to the 'Options menu' => 'Apollo Bar' and enable it.
